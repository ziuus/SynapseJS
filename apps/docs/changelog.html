Changelog

All notable changes to SynapseJS.

v0.3.0 Latest March 2026
New Features
  • Wave 3 tools: waitForElement, getPageUrl, setPageTitle, openModal, downloadFile, submitForm, checkboxToggle, setTheme
  • 3D Integration: interactWith3DScene for Spline designs
  • New hook: useSynapseSignals — handles all 20 signal types with one import
  • Showcase Gallery: 4 high-fidelity demos (Shopping, SaaS, 3D, Onboarding)
  • Production documentation platform at apps/docs
Improvements
  • Server/client split: hooks now in @synapsejs/core/client — zero SSR errors
  • ToolRegistry.unregister(), has(), and list() methods added
  • Rebranded from AxonJS → SynapseJS with npm-available scoped package @synapsejs/core
v0.2.0 February 2026
New Features
  • Wave 2 tools: scrollTo, copyToClipboard, toggleElement, selectDropdown, highlightElement
  • Multi-provider support: Groq, Google Gemini, OpenAI
  • Gemini 2.0 Flash integration with correct tool-call format
v0.1.0 January 2026
Initial Release
  • Wave 1 tools: interactWithScreen, readScreenText, observeState, fillForm, navigateTo, showNotification
  • Core Agent class with Groq provider
  • useSynapseDOM hook for DOM snapshotting
  • Monorepo structure with @synapsejs/core and demo app
