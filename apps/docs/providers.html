Providers

Connect SynapseJS to any LLM in seconds.

Overview

SynapseJS ships with first-class support for three LLM providers. You configure your provider when constructing the Agent. There is no additional setup — just pass your API key and you're done.

⚡
Groq
Default. Ultra-fast inference. Best for real-time tool calls.
llmProvider: 'groq'
✨
Google Gemini
Best-in-class multimodal reasoning. Free tier available.
llmProvider: 'gemini'
🤖
OpenAI
GPT-4o and o-series models with advanced tool calling.
llmProvider: 'openai'

Configuration

agent setup
import { Agent } from '@synapsejs/core';

// Groq (default, fastest)
const agent = new Agent({
  llmProvider: 'groq',
  apiKey: process.env.GROQ_API_KEY,
  model: 'llama-3.3-70b-versatile', // optional
});

// Google Gemini
const agent = new Agent({
  llmProvider: 'gemini',
  apiKey: process.env.GEMINI_API_KEY,
  model: 'gemini-2.0-flash', // optional
});

// OpenAI
const agent = new Agent({
  llmProvider: 'openai',
  apiKey: process.env.OPENAI_API_KEY,
  model: 'gpt-4o', // optional
});
