Core Concept

Agent

The central orchestrator that connects your LLM to your UI.

What is an Agent?

The Agent class is the heart of SynapseJS. It wraps your chosen LLM provider and gives it access to all 20 built-in tools (plus any custom tools you register). When the user sends a message, the Agent processes it through the LLM, picks the right tools, and returns typed AxonSignal objects for your frontend to handle.

Constructor

Agent constructor
import { Agent } from '@synapsejs/core';

const agent = new Agent({
  llmProvider: 'groq',       // 'groq' | 'gemini' | 'openai'
  apiKey: 'sk-...',           // your API key
  model: 'llama-3.3-70b-versatile', // optional, provider default if omitted
  systemPrompt: '...',       // optional custom system prompt
});

Options

Option Type Required Description
llmProvider'groq' | 'gemini' | 'openai'YesThe LLM backend to use.
apiKeystringYesAPI key for the chosen provider.
modelstringNoModel ID. Defaults to provider's recommended model.
systemPromptstringNoOverride the default system prompt.

agent.run()

The main method. Processes a conversation and returns a text response plus any tool calls the LLM chose to emit.

app/api/chat/route.ts
const result = await agent.run(messages, domElements);

// result.text     — the AI's text reply
// result.toolCalls — array of AxonSignal objects to process

return Response.json({
  messages: [...messages, {
    role: 'assistant',
    content: result.text,
    toolCalls: result.toolCalls
  }]
});
