copyToClipboard

Copy any string of text to the user's clipboard.

Overview

Use copyToClipboard when the user asks to copy a link, a code snippet, an API key, or any text. The AI provides the string and the frontend uses the Clipboard API.

Signal
COPY_TO_CLIPBOARD
Wave
Wave 2 — Enhanced

Parameters

ParameterTypeRequiredDescription
textstringYesThe text to write to the clipboard.

Frontend Handler

app/page.tsx
COPY_TO_CLIPBOARD: async ({ text }) => {
  await navigator.clipboard.writeText(text);
  showToast('Copied to clipboard!', 'success');
},
