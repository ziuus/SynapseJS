readScreenText

Read the text content of any DOM element by its ID.

Overview

Use readScreenText when the user asks what something says, what the current count is, or any other question about displayed text. The AI passes the element's ID and the frontend reads the live DOM value.

Signal
READ_ELEMENT
Wave
Wave 1 — Core UI

Parameters

Parameter Type Required Description
elementIdstringYesThe data-synapse ID of the element to read text from.

Frontend Handler

app/page.tsx
READ_ELEMENT: ({ elementId }) => {
  const el = document.getElementById(elementId);
  const text = el?.textContent ?? '';
  // Send back to AI via a follow-up message
  addMessage({ role: 'system', content: `Element "${elementId}" says: "${text}"` });
},

Example Intent

User → AI
// User says: "How many items are in the cart?"
// AI calls:
readScreenText({ elementId: "cart-count" })
// Signal emitted: READ_ELEMENT { elementId: "cart-count" }
