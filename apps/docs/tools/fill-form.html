Wave 1 — Core UI

fillForm

Batch-fill multiple form fields in a single AI call.

Signal: FILL_FORM

When to Use

The user wants to auto-populate a form with data — e.g. "Fill in my details".

Parameters

NameTypeRequiredDescription
fields { elementId: string; value: string }[] required Array of element ID + value pairs to fill

Example

User says: "Fill the contact form with my name Alex and email alex@test.com"

Agent tool call 
agent calls fillForm({
  fields: [
    { elementId: "name-input", value: "Alex" },
    { elementId: "email-input", value: "alex@test.com" }
  ]
})

Frontend Handler

Handle this signal in your component using useAxonSignals:

Signal handler (TypeScript) 
case 'FILL_FORM': {
  const fields: { elementId: string; value: string }[] = args.fields || [];
  fields.forEach(({ elementId, value }) => {
    const el = document.getElementById(elementId) as HTMLInputElement;
    if (el) {
      el.value = value;
      el.dispatchEvent(new Event('input', { bubbles: true }));
    }
  });
  showToast(`Filled ${fields.length} field(s)`, 'success');
  break;
}
