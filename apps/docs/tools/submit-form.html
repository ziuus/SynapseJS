submitForm

Programmatically submit a form element by its ID.

Overview

Use submitForm as the final step of an AI-assisted form-filling workflow. After calling fillForm to populate the fields, submitForm triggers the submission — including validation via browser-native requestSubmit() so required fields and input patterns are enforced.

Signal
SUBMIT_FORM
Wave
Wave 3 — Power

Parameters

Parameter Type Required Description
formIdstringYesThe id attribute of the <form> element to submit.

Frontend Handler

app/page.tsx
SUBMIT_FORM: ({ formId }) => {
  const form = document.getElementById(formId) as HTMLFormElement;
  if (form) {
    // requestSubmit() triggers validation, submit() bypasses it
    if (form.requestSubmit) { form.requestSubmit(); }
    else { form.submit(); }
    showToast('Form submitted!', 'success');
  }
},

Typical Workflow

Multi-step AI flow
// 1. AI fills the form fields
fillForm({ fields: [
  { elementId: 'name-input', value: 'Alice' },
  { elementId: 'email-input', value: 'alice@example.com' },
]})

// 2. AI submits the form
submitForm({ formId: 'signup-form' })
