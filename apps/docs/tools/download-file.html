downloadFile

Trigger a browser file download from any URL.

Overview

Use downloadFile when the user asks to download an export, a report, a CSV, or any binary asset. The frontend creates a temporary anchor element and triggers a click.

Signal
DOWNLOAD_FILE
Wave
Wave 3 — Power

Parameters

Parameter Type Required Description
urlstringYesThe URL of the file to download. Can be an API route, blob URL, or external link.
filenamestringNoSuggested filename for the saved file, e.g. "report-2026.csv".

Frontend Handler

app/page.tsx
DOWNLOAD_FILE: ({ url, filename }) => {
  const a = document.createElement('a');
  a.href = url;
  a.download = filename ?? '';
  document.body.appendChild(a);
  a.click();
  document.body.removeChild(a);
},
