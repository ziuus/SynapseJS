getPageUrl

Read the current page URL so the AI knows where it is.

Overview

Triggers the frontend to read window.location.href and feed it back to the AI as a system message. Takes no parameters.

Signal
GET_PAGE_URL
Parameters
None

Frontend Handler

app/page.tsx
GET_PAGE_URL: () => {
  const url = window.location.href;
  addMessage({ role: 'system', content: `Current URL: ${url}` });
},
