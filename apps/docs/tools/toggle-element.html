toggleElement

Show or hide any DOM element by its ID.

Overview

Use toggleElement when the user asks to hide a panel, show a section, collapse a menu, or toggle any visibility. If no visible value is provided, the current state is toggled.

Signal
TOGGLE_ELEMENT
Wave
Wave 2 — Enhanced

Parameters

ParameterTypeRequiredDescription
elementIdstringYesThe ID of the element to toggle.
visiblebooleanNotrue to show, false to hide. Omit to toggle current state.

Frontend Handler

app/page.tsx
TOGGLE_ELEMENT: ({ elementId, visible }) => {
  const el = document.getElementById(elementId);
  if (!el) return;
  const show = visible !== undefined ? visible : el.style.display === 'none';
  el.style.display = show ? '' : 'none';
},
