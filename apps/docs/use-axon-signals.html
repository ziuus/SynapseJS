useSynapseSignals

Handle every AI signal with a single, typed hook.

Overview

useSynapseSignals is the recommended way to wire up your entire signal pipeline. You pass a handler map — one callback per signal type — and the hook returns a processSignals function that you call after each AI response. Each callback is fully typed via the SynapseSignal discriminated union.

Usage

app/page.tsx
'use client';
import { useSynapseSignals } from '@synapsejs/core/client';
import { toast } from 'sonner';

const { processSignals } = useSynapseSignals({
  SHOW_NOTIFICATION: ({ message, type }) => toast[type](message),
  NAVIGATE: ({ url }) => router.push(url),
  SET_THEME: ({ theme }) => setTheme(theme),
  OPEN_MODAL: ({ elementId }) => openModal(elementId),
  FILL_FORM: ({ fields }) => fields.forEach(({ elementId, value }) => {
    const el = document.getElementById(elementId) as HTMLInputElement;
    if (el) el.value = value;
  }),
});

// After getting an AI response:
const data = await fetch('/api/chat', { ... }).then(r => r.json());
processSignals(data.messages.flatMap(m => m.toolCalls ?? []));
